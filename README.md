# Bolt KS

E-commerce site for Bang & Olufsen products in Kosovo.

**Domain:** bolt-ks.com

## Tech Stack
- Next.js 16
- React 19
- Tailwind CSS 4
- Prisma (database)
- Stripe (payments)

## Features (Planned)
- Product catalog with B&O speakers, headphones, TVs
- Shopping cart & checkout
- User accounts
- Order management
- Albanian/English support

## Development

```bash
npm install
npm run dev
```
